# LibraLingo 🤟

Uma aplicação educacional para aprendizado de LIBRAS (Língua Brasileira de Sinais) através de quizzes interativos.

## 📋 Sobre o Projeto

LibraLingo oferece uma forma divertida e interativa de aprender sinais básicos em LIBRAS, com 5 níveis de dificuldade progressiva:
- Nível 1: Cumprimentos Básicos  
- Nível 2: Conversas Cotidianas
- Nível 3: Família
- Nível 4: Alimentos
- Nível 5: Lugares

## 🚀 Tecnologias

- **Backend**: Java 21, Spring Boot 3.1.8, H2 Database
- **Frontend**: Angular 20, TypeScript 5.8, Standalone Components

## 💻 Como Executar

### Backend
\\\ash
cd backend
./mvnw spring-boot:run
\\\
Rodará em http://localhost:8080

### Frontend
\\\ash
cd frontend
npm install
npm start
\\\
Rodará em http://localhost:4200
